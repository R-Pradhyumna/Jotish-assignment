Contributing to Jotish-assignment

Thank you for considering contributing to Jotish-assignment!
Contributions of all kinds are welcome code, documentation, bug reports, feature requests, or ideas.

This guide is written to be beginner-friendly, so don’t worry if you’re new to Git or open source.

---

How You Can Contribute

You can help in many ways:

- Report bugs
- Suggest new features or improvements
- Fix bugs or implement features
- Improve documentation (README, comments, guides)
- Help review or discuss issues

---

Getting Started

1. Fork the Repository  
   Click the Fork button at the top-right of this repository.

2. Clone Your Fork  
   git clone https://github.com/<your-username>/Jotish-assignment.git  
   cd Jotish_assignment

3. Create a New Branch  
   git checkout -b my-feature-name

---

Making Changes

- Keep your changes focused and small
- Follow existing coding styles and patterns
- Add comments where the logic may not be obvious
- Test your changes if applicable

---

Committing Your Changes

Write clear and descriptive commit messages:

git commit -m "Fix bug in automation logic"

---

Submitting a Pull Request

1. Push your branch to your fork:  
   git push origin my-feature-name

2. Open a Pull Request against the main branch

3. Describe:
   - What you changed
   - Why you changed it
   - Any related issue numbers (if applicable)

---

Reporting Issues

When creating an issue, please include:

- A clear and descriptive title
- Steps to reproduce the problem
- Expected behavior
- Actual behavior
- Screenshots or logs (if helpful)

Good issue reports help us fix things faster.

---

Code Style Guidelines

- Keep code clean and readable
- Use meaningful variable and function names
- Follow the existing formatting in the project
- Avoid unnecessary complexity

If you’re unsure about style or structure, feel free to ask in an issue or PR.

---

Code of Conduct

Please be respectful and considerate in all interactions.
Harassment, discrimination, or toxic behavior will not be tolerated.

We want this to be a welcoming space for everyone.

---

Need Help?

If you’re stuck or unsure about something:

- Open an issue
- Ask a question in your pull request

We’re happy to help.

---

Thank You

Every contribution matters big or small.
Thanks for helping make Jotish-assignment better!
